GitHub on veebiplatvorm, kus saab koodi salvestada, redigeerida ja jälgida muudatusi ning projektide kallal töötada. Mina olen Valeria Balabolova, Kutsehariduskeskuse tudeng ja hakkan seda kasutama.
Ma kasutan GitHubi oma kooliprojektide ja kodutööde salvestamiseks.
See on kasulik ka klassikaaslastega koodi kallal koostöö tegemiseks.
GitHub Inc. asutasid 2008. aastal Tom Preston-Werner, Chris Wanstrath, PJ Hyett ja Scott Ciaccon.
GitHub on kasutajatele saadaval olnud alates 2008. aasta aprillist.
